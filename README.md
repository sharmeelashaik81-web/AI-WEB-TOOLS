# 🤖 AI Tools Hub

A modern and interactive web platform to discover the world's most powerful AI tools in one place.

This project provides a visually stunning UI where users can explore AI tools across multiple categories like Text AI, Image AI, Code AI, Audio AI, and Video AI.

---

## 🚀 Features

✨ Beautiful animated UI  
🎨 Glassmorphism card design  
📂 Category filtering (Text, Image, Code, Audio, Video)  
⚡ Smooth hover and tilt animations  
🌌 Animated background effects  
🚀 Direct launch buttons for each AI tool  
📱 Fully responsive design  

---

## 🧠 AI Tools Included

Some popular tools included:

- ChatGPT
- Claude
- Gemini
- DALL·E
- Leonardo AI
- GitHub Copilot
- ElevenLabs
- Runway ML
- Synthesia
- Perplexity

Users can click on any tool card to instantly open the official website.
